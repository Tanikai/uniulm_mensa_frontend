## Getting Started

### Prerequisites

- [pnpm](https://pnpm.io/)

### Installation

```bash
git clone https://github.com/Tanikai/uniulm_mensa_frontend.git
cd uniulm_mensa_frontend
pnpm i
```

### Running it locally

```bash
pnpm run dev
```
